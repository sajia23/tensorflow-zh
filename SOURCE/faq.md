### 如何安装 pip ？

 * Ubuntu (14.04)

        sudo apt-get update && sudo apt-get install -y python python-dev python-pip

 * CentOS 7

        yum update -y && yum install -y python python-devel epel-release.noarch python-pip

 * MACOS

        sudo easy_install pip
